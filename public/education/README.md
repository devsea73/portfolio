# Education Assets

This folder contains education-related images for the portfolio.

## Penn State University Logo

Place the Penn State University logo image here as `penn-state-logo.png`

**Required file:** `penn-state-logo.png`
**Recommended image specs:**
- Format: PNG with transparency
- Size: 80x80 pixels (will be displayed at this size)
- Should be the Penn State Nittany Lion shield logo

The image will be displayed in the Education section of the portfolio alongside the university information.
